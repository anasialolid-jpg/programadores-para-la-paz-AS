Al enviar una petición POST a http://localhost:3000/registro con el JSON de María, el servidor Express procesa los datos gracias al middleware express.json(), los extrae a través de req.body y devuelve una respuesta exitosa en formato JSON con un código de estado HTTP 200.

La respuesta exacta devuelta por el servidor es:
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}
